JSR-301 EL support

portletConfig: #{portletConfig} -->: object of type javax.portlet.PortletConfig

sessionPortletScope: #{sessionPortletScope} -->: mutable Map containing PortletSession attribute/values at PORTLET_SCOPE.

sessionApplicationScope: #{sessionApplicationScope} -->: mutable Map containing PortletSession attribute/values at APPLICATION_SCOPE.

portletPreferenceValue: #{portletPreferenceValue} -->: immutable Map containing the set of portlet preferences where the key is the name of the preference and the value is the first portlet preference value from the (potential) set of values.

portletPreferenceValues: #{portletPreferenceValues} -->: immutable Map containing the set portlet preferences where the key is the name of the preference and the values are the set of this portlet preference's values.
